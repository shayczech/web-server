Project Spotlight: CI/CD Pipeline

The Infrastructure as Code Stack

Fully Automated Terraform + Ansible DevSecOps Deployment

This site is running inside a Docker container on an AWS EC2 instance, managed end-to-end by the pipeline described below. Every update you see is a result of a Git push triggering a secure, auditable, and repeatable process.

1. Terraform (IaC)
  • ✓ Manages EC2, EIP, and Security Group.
  • ✓ S3 Remote State for concurrency control.
  • ✓ Outputs public IP used by Ansible.
2. Ansible (CM)
  • ✓ Installs Docker and cleans up old services.
  • ✓ Acquires and renews SSL Certs (Certbot/Route53).
  • ✓ Deploys Nginx container via Host Network mode.
3. CI/CD & Security
  • ✓ GitHub Actions orchestrates plan/apply/config.
  • ✓ Credentials secured via GitHub Secrets.
  • ✓ Least Privilege IAM policies used throughout.
