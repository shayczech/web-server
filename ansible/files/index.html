Project Spotlight: This CI/CD Pipeline

Deployed via GitHub Actions

Terraform + Ansible DevSecOps Stack

This site is running inside a Docker container on an AWS EC2 instance. The entire deployment process—from infrastructure creation to content update—is fully automated via a CI/CD pipeline. It features a two-container architecture: an Nginx proxy serving this page, and a Node.js API container providing the dynamic stats above.

1. Git Push (Code Change)
2. GitHub Actions (CI/CD Trigger)
3. Terraform & Ansible (Build & Configure)
4. AWS EC2 (Site is Live)
  • ✓ Terraform IaC: Manages EC2, EIP, and Security Group, with PoLP via IAM Role integration.
  • ✓ Ansible: Deploys containers and hardens the host with a UFW firewall configuration.
  • ✓ Nginx Proxy: Serves content, terminates SSL, and includes an API resilience check.
  • ✓ GitHub Actions: Orchestrates the entire build/deploy cycle from a single `git push`.
