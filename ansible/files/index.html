Terraform + Ansible DevSecOps Stack

This site is running inside a Docker container on an AWS EC2 instance. The entire deployment process—from infrastructure creation to content update—is fully automated via a CI/CD pipeline.

  • ✓ Terraform IaC: Manages EC2, EIP, and Security Group (S3 Remote State).
  • ✓ Ansible: Installs Docker, copies content, and forces Nginx restart using Handlers.
  • ✓ GitHub Actions: Orchestrates the entire plan/apply/config cycle.
  • ✓ Security: Uses GitHub Secrets and IAM Least Privilege policies.
