Secure DevSecOps Pipeline with GRC Compliance

This site demonstrates a comprehensive DevSecOps pipeline running on AWS infrastructure. The entire deployment process—from infrastructure creation to content update—is fully automated via GitHub Actions. It features a secure two-container architecture: an Nginx reverse proxy with SSL termination, and a Node.js API providing real-time GitHub statistics. The pipeline includes integrated security scanning, compliance mapping, and policy-as-code validation.
